# 贡献者公约

## 我们的承诺

身为社区成员、贡献者和领袖，我们承诺使社区参与者不受骚扰，无论其年龄、体型、可见或不可见的缺陷、族裔、性征、性别认同和表达、经验水平、教育程度、社会与经济地位、国籍、相貌、种族、宗教或性认同和性取向如何。

我们承诺以有助于建立开放、友善、多样化、包容、健康社区的方式行事和互动。

## 我们的准则

有助于为我们的社区创造积极环境的行为包括但不限于：

* 表现出对他人的同理心和善意
* 尊重不同的主张、观点和感受
* 给予并优雅地接受建设性反馈
* 承担责任并向因我们的错误而受影响的人道歉，从经验中学习
* 专注于不仅对我们个人，而且对整个社区最有利的事情

不当行为包括但不限于：

* 使用与性有关的言语或图像，以及任何形式的性注意或性暗示
* 挑衅、侮辱或贬损的评论，人身及政治攻击
* 公开或私下骚扰
* 未经明确许可，发布他人的私人信息，如住址或电子邮箱
* 其他有理由认定为违反职业操守的不当行为

## 责任和执行

社区领袖有责任解释和落实我们认可的行为准则，并妥善公正地对被视为不当、威胁、冒犯或有害的任何行为采取纠正措施。

社区领袖有权利和责任删除、编辑或拒绝与本行为准则不符的评论（comments）、提交（commits）、代码、维基（wiki）编辑、议题（issues）或其他贡献，并在适当时机告知做出审核决定的理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于当个人在公共场所正式代表社区时。代表我们社区的例子包括使用官方电子邮件地址、通过官方社交媒体帐户发帖，或在线上或线下活动中担任指定代表。

## 举报

如遇滥用、骚扰或其他不可接受的行为，请通过 1452492081@qq.com 向负责执行的社区领袖举报。所有投诉都将得到及时和公平的审查与调查。

所有社区领袖有义务尊重任何事件举报人的隐私和安全。

## 执行指导原则

社区领袖将遵循这些社区影响指导原则来判定违反本行为准则的后果：

### 1. 更正

**社区影响**：使用不恰当语言或其他被社区认定为不专业或不受欢迎的行为。

**后果**：社区领袖发出私人书面警告，明确说明违规性质，并解释行为为何不当。可能会要求公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列行为造成的违规。

**后果**：警告并对持续行为做出后果说明。在规定时间内，不得与相关人员进行任何互动，包括未经请求地与执行本行为准则的社区领袖互动。这包括避免在社区空间及外部渠道（如社交媒体）进行互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区准则，包括持续不当行为。

**后果**：在特定期限内，暂时禁止与社区进行任何形式的互动或公共交流。在此期间，不得与相关人员进行任何公开或私下互动，包括未经请求地与执行本行为准则的社区领袖互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：行为模式违反社区准则，包括持续不当行为、骚扰个人或对某类人进行攻击或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。
